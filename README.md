### Сделать перед стартом
- Убедиться, что версия node.js не ниже **16.13.0**
- `npm install`

### Запуск тестов
- Выполнить `npm run test`

### Для сборки проекта
- Выполнить `npm run build`

### Запуск проекта на localhost:3000
- Выполнить `npm run start`
